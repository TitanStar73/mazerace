# mazerace

This is a cool two player game!

![image](https://github.com/user-attachments/assets/d2b16a02-5f04-4b3f-965b-07bc5fd514f6)

# Objective

Beat your friend to the other side

# Controls
WASD to move red

IJKL to move blue

# Where to play?
Access it online through this [link](https://sprig.hackclub.com/share/JMcNVxChLZazCb9zlTiJ) on the sprig editor. 
Click Run dropdown
 - Run to play on browser
 - Run on Device to upload to your Sprig device!
